## 🚀 Overview

API Login System is a modern authentication project that simulates the core functionality of a real-world login system without using any frameworks.

Users can create an account, log in using their email or phone number, manage their profile, and stay logged in with the **Remember Me** feature powered by Cookies.

The project communicates with a REST API through **Fetch API**, performing complete CRUD operations while providing a modern Glassmorphism user interface.

---

## ✨ Features

- 🔐 User Registration (Sign Up)
- 🔑 Login with Email or Phone Number
- 🍪 Remember Me functionality using Cookies
- 👤 Personal Dashboard after authentication
- ✏️ Edit user information
- 🗑️ Delete user account
- 📡 REST API integration with MockAPI
- 🔄 Full CRUD Operations
- 🧠 Form validation using Regular Expressions
- ⚡ Dynamic UI updates without page reload
- 🎨 Modern Glassmorphism interface
- 📱 Responsive Design
- ✨ Smooth animations and interactive effects
